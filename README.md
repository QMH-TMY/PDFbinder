# PDFbinder & PDFcuter #
------------------------
Command line interface to merge pdf files and split pdf file.

# requirement #
	PyPDF2 
    $ sudo pip install PyPDF2

# Usage1 #
    $ python PDFbinder.py s1.pdf s2.pdf ... output.pdf

# Usage2 #
    $ python PDFcuter.py input.pdf output.pdf 1 4
