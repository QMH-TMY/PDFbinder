PDFbinder
=============
Command line interface to merge pdf files.

PDFcuter
=============
Command line interface to split a pdf file.

Versions
--------
PDFbinder works with Python 2.7,3

Requirement
-----------
PyPDF2 

Usage
-----

::

    $ PDFbinder s1.pdf s2.pdf... output.pdf
    $ PDFcuterr s1.pdf output.pdf start end
